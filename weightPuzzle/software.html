weightPuzzle.ino Main State Diagram Initializes the GPIOs for the load sensors, relay, LED, and button.
Tares the load sensors to start reading at 0 lbs.
Allows the administrator to set the TargetWeights.
During game play, it reads the different load sensors and determines when to unlock the magnetic lock.
HX711.h HX711.c This is github user Bogde's HX711 library for the HX711. It is released under a GNU GENERAL PUBLIC LICENSE
The HX711 chip reads load cells very well. The Sparkfun breakout board is compatible with any wheatsone bridge based load cell.
The HX711 board can be powered from 2.7V to 5V
