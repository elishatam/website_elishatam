Play Game

Description of Functionality

Project Requirements 		FLICKER - Fast, Laughable, Interactive, Computerized, Kitsch using Electromechanical Robots
Goal: Design and build an interactive electro-mechanical depiction of an easily recognizable movie scene.

Interaction
  • The user must interact with the scene to progress it.
  • The game must have at least 3 distinct sequential user interactions.
    • One must be an analog input from the user.
    • One must involve non-contact sensing.
Timing
  • The FLICKER should take at least 20 seconds to 45 seconds to complete.
  • If the user is unable to successfully complete the game within 45 seconds, the FLICKER should stop interacting with the user and automatically trigger the next FLICKER in the chain.
  • The FLICKER should include a creative display of the passage of time.
Connection between FLICKERs
  • Each FLICKER must be capable of being initially triggered by another FLICKER and at the end of the interaction must provide the trigger for the next FLICKER in the chain
  • The FLICKER must have a force start button.
Success and Fail Indicators
  • When the user successfully completes the game, the FLICKER should provide an exciting audio-visual indication of success and trigger the next machine in the chain.
  • If the 45 second time limit expires before the user successfully completes the game, the display of success should not be activated.
