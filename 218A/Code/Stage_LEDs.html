Stage_LEDs Module

This software module

  1) Controls the 3 Stage LEDs which is driven by a 4 line decoder

 

Public Functions:

resetStage

  Set output to decoder to zero which corresponds to an output of the decoder that is NOT connected

  to an LED

 

activateStage(myStage)

  Set output to the decoder corresponding to the LED associated with the input myStage (which is

  one of four options:  STAGE_IDLE, STAGE_IR, STAGE_GRAPPLE, STAGE_DOWNLOAD with STAGE_IDLE

  corresponding to all LEDs off)
