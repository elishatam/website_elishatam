Project Requirements



For our FLICKER project, we reenacted the scene from Mission: Impossible where Tom Cruise sneaks into the CIA headquarters to download the file of undercover agents. In our game, the user must complete 3 tasks:
  • Task 1: Disable the security system by breaking the IR beam.
  • Task 2: Grapple down in range of the computer.
  • Task 3: Insert the floppy disk to download the file, and eject the disk to return to safety.
If the user completes all 3 tasks before the CIA analyst returns from the bathroom, the user wins!
